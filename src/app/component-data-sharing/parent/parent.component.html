Sharing Data Between Components:

Parent to Child - by @Input :

When you declare a variable with the Input decorator in the child component, it allows that variable to be received from a parent template.

Child to parent - by @ViewChild :

ViewChild allows one component to be injected into another, giving the parent access to its attributes and functions.
{{ ChildMessageByViewChild }}

Child to parent - by @Output & Event Emitter :

Another way to share data is to emit data from the child, which can be listned to by the parent. This approach is ideal when you want to share data changes that occur on things like button clicks, form entires, and other user events.
